-----------Line Break----------
The spacing beteen code in an html file doesn't affect the positioning of elements in the browser.
Modifying the spacing in the browser we can use html's line break element 
i.e <br>

The line break element is unique because it is only composed of a starting tag. 
We can use it anywhere within our HTML code and a line break will be shown in the browser.

The Nile River is the longest river

kilometers long (approximately 4,260
miles).
